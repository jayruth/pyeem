# pyeem
Python Library for Processing EEM Spectra

## pyeem functions 
#### (under development)
* Import meta-data
* Import Spectra from instument file(s)
   - EEMs
   - Absorbance Measurements
* Subtract blanks
* Remove Scatter
  - Using interpolation
  - Replace with fixed value (i.e. zero)
* Ramam normalize
* Crop EEM
* Inner filter effect correction
* **EEM Vizualization**
